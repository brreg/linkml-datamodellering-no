# dqv-core 

DQV-kjerneklassar og -slots utan referanse til dcat-ap-no. Denne fila vert importert av dcat-ap-no for å gje tilgang til Kvalitetsmerknad og Kvalitetsmaaling på Datasett, utan å skape sirkulær import. dqv-ap-no importerer dcat-ap-no og narrowar har_maal.range via slot_usage.

URI: https://data.norge.no/ap-no/dqv-core
